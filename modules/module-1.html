📘 Pengujian Penetrasi Jaringan

Minggu 1-2 | 10 jam praktik | Tags: #Networking #Security #Metasploit

📌 Tujuan Modul

Memahami dan mempraktikkan metodologi ethical hacking dari awal hingga akhir pada lingkungan yang terkendali.

🛠️ Alat yang Digunakan

  • Kali Linux (VM)
  • Nmap / masscan
  • Metasploit Framework
  • Burp Suite (opsional)
  • VirtualBox (target Windows/Linux)

📋 Langkah-Langkah Metodologi

  1. Reconnaissance (OSINT) – mengumpulkan informasi target tanpa interaksi langsung.
  2. Scanning – menggunakan Nmap untuk menemukan port terbuka, service, dan OS.
  3. Vulnerability Assessment – mencocokkan service dengan CVE yang diketahui.
  4. Exploitation – menggunakan Metasploit untuk mendapatkan akses.
  5. Post Exploitation – mengumpulkan bukti, privilege escalation, dan persistence.
  6. Reporting – menyusun laporan temuan dan rekomendasi perbaikan.

💡 Contoh Mock Data (hasil scanning)

{
  "target": "192.168.1.10",
  "open_ports": [22, 80, 443, 3306, 8080],
  "os_guess": "Ubuntu 20.04",
  "vulnerabilities": [
    "CVE-2021-41773 (Apache Path Traversal)",
    "CVE-2022-22965 (Spring4Shell)"
  ],
  "exploit_status": "success",
  "access_gained": "www-data shell"
}

📝 Catatan Pribadi

Pada percobaan pertama, exploit terhadap Apache berhasil mendapatkan reverse shell. Namun setelah patching, serangan tidak berhasil. Ini menunjukkan pentingnya manajemen patch.
